# Dynamic_Supermarket_Sales_Dashboard
This project delivers an interactive Excel Sales Dashboard designed to give retail managers and executives a clear, consolidated view of supermarket performance. Built entirely within Microsoft Excel, the dashboard transforms raw transactional sales data into dynamic, filterable visualizations covering revenue, profit, product performance, sales channels, and payment behavior. It enables rapid, data-driven decisions without the need for external BI tools.
